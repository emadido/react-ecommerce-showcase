# React E-commerce Product Showcase

## 🧠 Descrizione

Una semplice vetrina prodotti stile e-commerce che utilizza una API esterna per mostrare prodotti dinamici.

## 🛠️ Tecnologie

- React
- Vite
- Fetch API
- CSS

## ✨ Features

- Lista prodotti da API
- Loading state
- Gestione errori
- Responsive layout

## 📦 API utilizzata

https://dummyjson.com/products

## ▶️ Come avviare il progetto

```bash
npm install
npm run dev
```
